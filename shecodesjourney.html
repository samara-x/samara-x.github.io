About this portfolio

SheCodes Plus is a six-month, project-based program for women and non-binary coders moving from programming fundamentals into employable full-stack skills. This portfolio documents that progression - the Python foundations, the first HTML websites, the move into React, the Django back-ends, and ultimately a live full-stack crowdfunding platform and a team-built collaborative bucket list app presented to industry mentors.

The learning journey

Each month built on the last - fundamentals, then structure, then interactivity, then a back-end, then a full application, then a production deployment with a team.

Months 1–2 · Python Foundations

Variables, control flow, functions, and data structures

Core Python through guided exercises: variables, conditionals, loops, functions, lists, dictionaries, file I/O, and object-oriented programming — culminating in the Plus Weather Project (CSV parsing, date formatting, temperature statistics) validated against a full unittest suite.

Month 2 · The Web, From Scratch

HTML, CSS and a first deployed site

Built and shipped a personal portfolio site (samara-x.github.io) using semantic HTML, responsive CSS, and custom styling — the first public proof-of-life as a developer.

Month 3 · JavaScript & React

From vanilla JS to component-based UIs

JavaScript fundamentals (arrays, functions, DOM) followed by the React ecosystem: Vite, JSX, props, hooks, state management, and React Router. Built a React message app, a todo list, and a multi-page router tutorial.

Month 4 · Django REST Framework

A real back-end with auth, permissions, and a PostgreSQL database

Designed an API for a crowdfunding platform: custom user model, token authentication, fundraisers, pledges, nested serializers, permission classes, custom 404s, and deployed to Heroku with PostgreSQL.

Month 5 · Full-Stack: FundingFour

React frontend meets Django backend — in production

Wired a React + Vite frontend to the Django REST API: token auth via localStorage, custom hooks (useAuth, useFundraiser), a responsive UI, and a deployed site on Netlify backed by the Heroku API.

Month 6 · Capstone Group Project: KICKIT

Built and pitched in a team of developers

"Owl-gorithms" team delivered KICKIT — a collaborative bucket-list platform with JWT auth, Google OAuth, roles & permissions, optimistic voting, invites, notifications and accessibility baked in. React 19, Tailwind v4, Django 5, PostgreSQL. Presented to industry mentors and peers.

Featured projects

The three capstone deliverables — each an attempt at a production-grade full-stack application.

Capstone · Group Project

KICKIT — Collaborative Bucket List

Team: Owl-gorithms · Role:Product Designer

A team-built platform that transforms "we should do that sometime" into "lock it in." Friends, families and teams create shared bucket lists, propose items, vote optimistically, and lock in what to do next - without relying on social media.

React 19 Vite 7 Django 5 DRF + SimpleJWT PostgreSQL Tailwind v4 Google OAuth

Highlights: JWT + Google OAuth, role-based permissions (Owner / Editor / Viewer), optimistic vote UI with rollback, polled notifications, invite-link flow, emoji reactions, calendar export, accessibility-first UI.

Full-Stack · Solo

FundingFour — Crowdfunding Platform

Role: Solo full-stack build (React frontend + Django REST backend)

A lightweight, human-centred crowdfunding platform for small-crowd support — pledging time, commitment or presence rather than money. Built on a React frontend and a Django REST Framework backend with token authentication and PostgreSQL.

React 18 Vite React Router Django REST Framework Token Auth PostgreSQL Netlify Heroku

Highlights: Sign-up + token-auth login, create / view / delete fundraisers, pledge with optional anonymity and comment, progress bar & percentage complete, responsive design, custom 404, deployed frontend (Netlify) + backend (Heroku).

PRIMAL
Full-Stack · Solo Extension

PRIMAL — Calisthenics Mastery

Role: Solo full-stack build (self-directed extension project)

A full-stack calisthenics progression app. Users browse skills, track sessions, build workouts, mark steps as mastered, and sync progression across devices once logged in. Anonymous users get a localStorage experience; authenticated users sync to PostgreSQL.

React 18 Vite Tailwind CSS Django 5 DRF + SimpleJWT PostgreSQL Axios

Highlights: JWT auth (access + refresh), custom user model, sessions & stats aggregation endpoint, saved workouts, progression API that syncs localStorage to Postgres on login, dark/light theming via Context.
